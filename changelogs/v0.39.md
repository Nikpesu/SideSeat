# SideSeat v0.39

Datum izdanja: 2026-06-17

Autići u pozadini sada se kreću zadanom brzinom i pouzdano prate prave ceste.

## Changed

- Autići se kreću **30 km/s** po stvarnoj duljini rute (umjesto fiksnih 5 sekundi po ruti).

## Fixed

- Pozadinske rute pouzdano koriste **pravu OSRM cestovnu geometriju**: zahtjevi su staggerani i ponavljaju se (do 4 pokušaja s odgodom), pa rute više ne ostaju ravne crte kad je prvi zahtjev odbijen ili istekne. Vanjska (halo) linija ažurira se zajedno s rutom.

## Docker

- Docker Hub tagovi: `nikolica/sideseat:v0.39` i `nikolica/sideseat:latest`.
- Image je namijenjen platformi `linux/amd64`.
