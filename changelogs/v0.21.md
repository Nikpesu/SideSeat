# SideSeat v0.21

Datum izdanja: 2026-06-11

Ovo izdanje omogućuje korištenje SideSeat AI asistenta preko Open WebUI-ja ili izravno preko DeepSeek API-ja.

## Added

- Dodan je konfigurabilni AI provider tip `OpenWebUi` ili `DeepSeek`.
- Dodani su integracijski testovi za models i chat endpointove oba providera.
- Dodani su Docker i README primjeri za izravnu DeepSeek konfiguraciju.

## Changed

- AI konfiguracija sada koristi zajedničke varijable `AI_API_TYPE`, `AI_BASE_URL`, `AI_API_KEY` i `AI_MODEL`.
- Chat i models putanje automatski se prilagođavaju odabranom provideru.
- AI sučelje prikazuje naziv aktivnog providera.
- Cijeli korisnički i aplikacijski AI kontekst šalje se kao jedan strukturirani JSON system message.

## Security

- API ključ ostaje isključivo na serveru i ne šalje se klijentskom JavaScriptu.
- AI kontekst zadržava postojeća ograničenja za tajne i osjetljive podatke.

## Docker

- Docker Hub tagovi: `nikolica/sideseat:v0.21` i `nikolica/sideseat:latest`.
- Dockerfile labela i zadani image u `docker-compose.hub.yml` ažurirani su na `v0.21`.
- Compose konfiguracija podržava izbor Open WebUI ili DeepSeek providera bez izmjene imagea.
