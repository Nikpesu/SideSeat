# SideSeat v0.26

Datum izdanja: 2026-06-15

Ovo izdanje dodaje stvarne OpenStreetMap prikaze ruta, geocoding gradova i sigurnije zajedničke write tokove za UI, REST, AI i MCP.

## Added

- Lokalno poslužen Leaflet 1.9.4 s interaktivnim OpenStreetMap kartama na početnoj te detaljima vožnji i rezervacija.
- Animirani približni prikaz rute s crvenim polazištem, zelenim odredištem, light/dark obradom pločica i reduced-motion podrškom.
- Hover i keyboard preview ruta na desktopu te mobilni gumb za prikaz rute.
- Nullable koordinate gradova, migracija i backfill koordinata za 22 postojeća seed grada.
- Nominatim geocoding s ručnim overrideom, 30-dnevnim cacheom i ograničenjem na jedan zahtjev u sekundi.
- AI i MCP prepare/confirm/cancel write alati za gradove, vozila, vožnje, rezervacije i ocjene.
- Globalna role-aware pretraga, audit pregled, correlation ID, strukturirani request logging te live/ready health endpointi.
- Integracijski testovi za navigaciju, karte, geocoding, AI/MCP confirmation tokove i SQL Server ograničenja.

## Changed

- Kreiranje grada kroz MVC, REST, AI i MCP koristi ista geocoding i poslovna pravila.
- Početni Vozni red koristi trajnu mapu koja mijenja rutu bez ponovnog stvaranja Leaflet instance tijekom AJAX pretrage.
- Navigacija koristi dvoredni desktop navbar i scrollabilni role-aware offcanvas sidebar.
- Početna stranica, kartice i hover efekti prilagođeni su responsive prikazu bez odsijecanja sadržaja.
- CRUD i rezervacijski write tokovi koriste stroža pravila vlasništva, statusa, kapaciteta i relacijskih ograničenja.

## Fixed

- Spriječeno višestruko izvršavanje, potvrđivanje tuđih ili isteklih AI/MCP akcija.
- Legacy gradovi bez koordinata prikazuju kontrolirani fallback bez JavaScript pogreške.
- Produkcijske migracije imaju retry, a API poslovne i konkurentne greške koriste standardizirane HTTP statuse.

## Security

- MCP endpoint zahtijeva Bearer ključ i izvršava alate u kontekstu konfiguriranog servisnog korisnika.
- Audit i request logging izbjegavaju zapisivanje lozinki, ključeva, kartičnih podataka i osjetljivih identifikatora.
