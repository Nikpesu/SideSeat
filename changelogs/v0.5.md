# SideSeat v0.5

Datum izdanja: 2026-06-07

Izdanje objedinjuje sve preglede voznji pod jednim glavnim tabom i uvodi pregledniji subselector.

## Added

- Subselector unutar taba `Voznje` za brzo prebacivanje izmedu dostupnih pregleda.
- Pregledi `Sve voznje`, `Sve aktivne voznje`, `Sve gotove voznje`, `Moje sve voznje` i `Moje aktivne voznje`.
- Integracijski test koji provjerava vidljivost admin opcija i zabranu pristupa ostalim korisnicima.

## Changed

- Glavna navigacija sada prikazuje jedan tab `Voznje` umjesto odvojenih tabova.
- Stari endpoint aktivnih voznji preusmjerava na objedinjeni pregled.
- Admin vidi sve i gotove voznje, dok ostali korisnici vide javne aktivne i vlastite preglede.
