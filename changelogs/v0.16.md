# SideSeat v0.16

Datum izdanja: 2026-06-10

SideSeat dobiva potpuno novi AI-native vizualni identitet i ugrađeni AI Copilot povezan s privatnim Open WebUI servisom.

## Added

- Globalni Performative UI inspirirani dizajn s aurorom, glass površinama, gradient tipografijom, marquee elementima i animiranim metrikama.
- SideSeat AI Copilot dostupan u donjem desnom kutu na svim stranicama.
- Server-side Open WebUI gateway s automatskim odabirom dostupnog modela.
- Kontekst trenutne stranice, povijest razgovora i brze početne AI naredbe.
- Docker varijable `OPENWEBUI_API_KEY` i opcionalni `OPENWEBUI_MODEL`.

## Changed

- Redizajnirani su globalni layout, navigacija, naslovnica, kartice, tablice, forme, selektori i gumbi.
- README sada dokumentira AI arhitekturu i sigurnu lokalnu/Docker konfiguraciju.

## Security

- Open WebUI API ključ ostaje na serveru i nikada se ne šalje klijentskom JavaScriptu.
- AI endpoint koristi antiforgery zaštitu, validaciju ulaza, ograničenu povijest i rate limiting.
