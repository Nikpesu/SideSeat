# SideSeat v0.23

Datum izdanja: 2026-06-11

Ovo korektivno izdanje popravlja formatiranje ruta i poveznica u odgovorima SideSeat AI asistenta.

## Changed

- AI alati uz URL sada vraćaju i gotove Markdown poveznice za detalje vožnji i rezervacija.
- Lista vožnji vraća poveznicu na odgovarajući pregled prema opsegu i statusu.
- System uputa izričito zahtijeva obični znak `→` i klikabilne interne poveznice.

## Fixed

- LaTeX zapisi poput `$\rightarrow$` više se ne prikazuju korisniku.
- Backend normalizira česte LaTeX oznake smjera u znak `→`.
- Tekst `Detalji` sada se ispisuje kao stvarna Markdown poveznica kada alat vrati URL.
