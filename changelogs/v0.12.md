# SideSeat v0.12

Datum izdanja: 2026-06-08

Izdanje uvodi sigurniji Docker start bez demo podataka po defaultu, provjeru salda prije rezervacije i mock checkout za nadoplatu računa.

## Added

- Custom file picker za slike recenzija s gumbom 'Odaberi slike' koji poštuje dark i light temu.
- Mock checkout stranica za uplatu sredstava s opcijama Kartica, PayPal i Revolut Pay.
- Toast upozorenje za nedovoljan saldo s izračunom manjka i gumbom 'Uplati sredstva'.
- Sigurno spremanje mock kartice i adrese plaćanja; spremaju se samo maskirani kartični podaci.

## Changed

- Docker Compose sada koristi DUMMY_DATA=false po defaultu, a demo podaci se uključuju samo eksplicitno.
- Rezervacija vožnje provjerava ukupno potrebna sredstva: nova rezervacija plus već zakazane rezervacije.
- README i lab5Doc ažurirani su za predaju i novo ponašanje Docker/demo podataka.

## Fixed

- Bijeli native choose-files gumb više ne probija dark temu.
- Docker baza bez DUMMY_DATA=true više ne ostavlja migracijske demo zapise na prvom pokretanju.
