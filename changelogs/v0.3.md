# SideSeat v0.3

Datum izdanja: 2026-06-07

Izdanje dodaje slike uz recenzije, poboljsava prijavu i privatnost korisnickih profila te uvodi pouzdaniji Docker release workflow.

## Added

- Upload do pet slika uz ocjenu voznje, uz spremanje relativne putanje u bazi.
- Thumbnail prikaz slika na svim prikazima recenzija i modalno uvecanje klikom.
- Google gumb za prijavu i registraciju s Google ikonom.
- Integracijski testovi za slike recenzija, pristup profilima i `RememberMe` prijavu.
- Projektni `version-control-dockerhub` skill za verzioniranje i Docker Hub objavu.

## Changed

- Privitci su premjesteni s voznji na ocjene kroz model `OcjenaSlika`.
- Detalje korisnika moze vidjeti samo administrator ili vlasnik profila.
- Docker Compose i image koriste verzijske OCI i SideSeat labele.
- Docker Hub release objavljuje verzionirani tag i `latest`.

## Fixed

- Popravljen login s ukljucenom opcijom `Zapamti me`.
- Uklonjen stari upload datoteka s forme za uredivanje voznje.
